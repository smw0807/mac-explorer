# Mac Explorer

윈도우 탐색기 스타일의 macOS용 파일 관리자 (Electron + React)

## 주요 기능

### 탐색

- **폴더 트리 사이드바** — 탐색기처럼 왼쪽에서 폴더 구조를 펼쳐 탐색
- **즐겨찾기 사용자화** — 폴더를 '즐겨찾기' 섹션에 드래그해서 추가, 우클릭으로 제거 (영구 저장)
- **주소창** — breadcrumb 클릭 이동 + 직접 경로 입력 (`⌘L`, `~` 지원)
- **탭** — `⌘T` 새 탭, `⌘W` 닫기, 폴더 우클릭 → 새 탭에서 열기
- **듀얼 패널** — `⇧⌘D` 로 화면 2분할, 패널 간 복사/이동 편리
- **세션 복원** — 종료 시 탭 구성(경로·듀얼 패널·숨김 파일)을 저장하고 재시작 시 그대로 복원
- **타이핑 점프** — 목록에서 파일명 첫 글자를 입력하면 해당 항목으로 즉시 선택 이동

### 보기

- **상세 보기 정렬** — 이름 / 수정한 날짜 / 유형 / 크기 컬럼 클릭 정렬
- **아이콘 보기 / 썸네일 그리드** — 툴바 ⊞ 버튼으로 전환, 이미지 파일은 실제 썸네일 표시
- **네이티브 파일 아이콘** — macOS 시스템 아이콘 그대로 표시
- **미리보기** — `Space` 로 이미지·동영상·오디오·텍스트 미리보기 패널
- **Quick Look** — `⌘Y` 로 시스템 미리보기 (PDF·오피스 문서 등도 지원)
- **숨김 파일 표시** — `⇧⌘.`

### 파일 작업

- **잘라내기 / 복사 / 붙여넣기** — `⌘X` / `⌘C` / `⌘V` (파인더에 없는 그 기능!)
- **드래그 & 드롭** — 폴더 행·다른 패널·사이드바로 끌어서 이동, `⌥` 누르면 복사, Finder에서의 드롭도 지원
- **진행률 표시 복사** — 대용량 복사 시 진행률 바 + 취소 버튼, 취소하면 부분 파일 자동 정리
- **이름 충돌 처리** — 같은 이름이 있으면 둘 다 유지 / 덮어쓰기 / 건너뛰기 / 취소 선택
- **실행 취소** — `⌘Z` 로 이동·복사·이름변경·생성 작업 되돌리기
- **일괄 이름 변경** — 다중 선택 후 찾기/바꾸기 또는 연번 패턴(`휴가-###`)으로 한 번에 변경
- **ZIP 압축/해제** — 우클릭 메뉴로 압축하기·압축 해제 (macOS 내장 zip/ditto 사용)
- **다음으로 열기** — 기본 앱이 아닌 다른 앱을 골라서 열기
- **삭제는 휴지통으로** — `⌘⌫` (바로 삭제되지 않아 안전)

### 검색 · 갱신

- **검색** — 입력 즉시 현재 폴더 필터, `Enter` 시 하위 폴더까지 재귀 검색
- **Spotlight 검색** — `⌘Enter` 로 파일 이름 + 내용까지 인덱스 기반 즉시 검색
- **자동 새로 고침** — 현재 폴더를 `fs.watch`로 감시해 외부 변경(생성·삭제·이름변경)을 즉시 반영

## 단축키 요약

| 키 | 동작 |
|---|---|
| ⌘C / ⌘X / ⌘V | 복사 / 잘라내기 / 붙여넣기 |
| ⌘Z | 실행 취소 |
| ⌘A | 전체 선택 |
| F2 | 이름 바꾸기 |
| ⌘⌫, Delete | 휴지통으로 이동 |
| Enter, ⌘↓ | 열기 |
| ⌘↑ | 상위 폴더 |
| ⌘L | 주소창 편집 |
| ⌘F | 검색 (Enter: 재귀 검색, ⌘Enter: Spotlight) |
| ⌘R | 새로 고침 |
| ⌘T / ⌘W | 새 탭 / 탭 닫기 |
| ⇧⌘D | 듀얼 패널 토글 |
| ⇧⌘N | 새 폴더 |
| ⇧⌘. | 숨김 파일 표시 |
| Space | 미리보기 토글 |
| ⌘Y | Quick Look |
| ↑ ↓ ← → | 항목 이동 (그리드에서는 ↑↓가 열 단위) |
| 아무 글자 | 해당 글자로 시작하는 항목으로 점프 |

## 실행 방법

```bash
# Node.js 18+ 필요 (https://nodejs.org)
cd mac-explorer
npm install

# 개발 모드 (핫 리로드)
npm run dev

# 프로덕션 실행
npm run start

# .dmg 배포 파일 만들기 (release/ 폴더에 생성)
npm run dist
```

첫 실행 시 macOS가 홈/문서/다운로드 폴더 접근 권한을 물어보면 허용해 주세요.
(시스템 설정 → 개인정보 보호 및 보안 → 파일 및 폴더)

## 구조

```
electron/main.cjs      # 메인 프로세스: 창 생성 + 파일시스템 IPC
                       #   (복사 진행률, fs.watch, zip, Spotlight, 설정 저장 등)
electron/preload.cjs   # contextBridge로 안전하게 API 노출
src/App.jsx            # 탭 / 듀얼 패널 / 전역 단축키 / 세션 저장·복원 / 즐겨찾기
src/components/
  Sidebar.jsx          # 즐겨찾기 + 폴더 트리 (드롭 타깃)
  Pane.jsx             # 주소창, 파일 목록(리스트/그리드), 검색, 미리보기, 컨텍스트 메뉴
  ContextMenu.jsx      # 우클릭 메뉴
  FileIcon.jsx         # 네이티브 파일 아이콘 (확장자 단위 캐시)
  BatchRenameDialog.jsx # 일괄 이름 변경 모달
src/dnd.js             # 드래그 & 드롭 + 이름 충돌 처리 공용 로직
src/undo.js            # 파일 작업 실행 취소 스택
src/util.js            # 정렬, 아이콘, 포맷 유틸
```

앱 설정(세션·즐겨찾기)은 `~/Library/Application Support/Mac Explorer/settings.json`에 저장됩니다.
